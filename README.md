# Дыхание Шинигами (死呼吸)

Веб-сайт о дыхании Шинигами - древнем боевом искусстве, основанном на связи между миром живых и миром мёртвых.

## Описание

Сайт посвящён дыханию Шинигами и включает информацию о:
- Описании дыхания и религии богов Изанами и Изанаги
- Составе и принципах техники
- Катах (формах) дыхания
- Системе повышения рангов

## Технологии

- HTML5
- CSS3

## Структура проекта

```
shinigami/
├── public/
│   ├── index.html       # Главная страница
│   ├── sostav.html      # Страница о составе дыхания
│   ├── katy.html        # Страница с катами (формами)
│   ├── sistema.html     # Страница о системе повышения
│   └── styles.css       # Стили сайта
├── README.md            # Документация проекта
└── .gitignore           # Файл для Git
```

## Запуск

Просто откройте `public/index.html` в любом современном браузере! Сайт полностью статический и не требует сервера.
